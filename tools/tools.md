# Tools Used in This Repository

| Tool | Description |
|-----|-------------|
| kubectl | Command line tool used to interact with Kubernetes clusters. |
| kind | Tool for running local Kubernetes clusters using Docker containers as nodes. |
| helm | Package manager for Kubernetes used to deploy applications using charts. |
| terraform | Infrastructure as Code tool used to provision and manage cloud resources. |
| k9s | Terminal-based UI that helps monitor and manage Kubernetes clusters. |
| lens | Graphical Kubernetes IDE used for visualizing and managing cluster resources. |
| docker | Container platform used to build and run containers. |
| zsh | Advanced shell used to enhance terminal productivity. |
| powerlevel10k | Zsh theme used to customize the terminal prompt and improve developer workflow. |
